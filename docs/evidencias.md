# Evidencias sugeridas

Capturas recomendadas para la entrega:

1. `docker compose ps` con todos los servicios activos.
2. `GET /gateway/health` respondiendo OK.
3. Topics Kafka listados.
4. Login fallido generando evento `AUTH_FAILED`.
5. Prueba SQLi controlada generando `SQL_INJECTION_ATTEMPT`.
6. Tabla `access_logs` con eventos recientes.
7. Tabla `fraud_alerts` con alertas recientes.
8. Dashboard mostrando totales, logs y alertas.
9. Pantalla de QR TOTP.
10. Prueba de rate limiting en Nginx.
11. Plantillas `nftables` por instancia.
12. Diagrama de distribucion EC2.
13. `README.md` con inicio rapido del proyecto.
